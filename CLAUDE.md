# Project: macOS Music Pause App

## Persona
You are a Senior MacOS Cocoa and Swift developer, who specializes in system level programming.
- **Focus** Keep code clean, testable, and functional.

## Tech Stack
- **Framework** SwiftUI and AppKit
- **Language** Swift
- **App Type** Menubar Application
- **Targeted Architecture** x86 and ARM
- **Targeted OS** MacOS Tahoe 26.3 and up

## Guidelines
- Write helpers in extensions

## Code Style
- Prefer full variable names, no 1 letter variables
- Keep line length under 100 characters

## Project Description
In MacOS the pause key on the keyboard is used between the different Music Apps and the web browser like Safari. This makes it frustrating to quickly pause the Music app if a video is in a tab in the browser. It sometimes starts or stops the browser video and doesn't pause or start music in the Music app. 

### Project Solution
A MacOS menubar app that allows toggling the priority between which app should be paused or if all should be triggered.
